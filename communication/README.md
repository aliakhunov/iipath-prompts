# Communication prompts

AI prompts for supporting professional communication in real work scenarios.

The focus is on clarity, structure, and intent rather than stylistic creativity.

---

## Use cases

Prompts in this directory are used for:
- preparing and structuring messages,
- clarifying positions and arguments,
- handling difficult or sensitive conversations,
- communication with stakeholders and teams.

---

## Language note

Most prompts are written in **Russian**.

This is intentional, as communication quality depends on language nuance and context.
